# 文本预处理
## 词嵌入
在自然语言系统中词是基本的单元，词语组成句子，句子再组成段落，段落最终再组成文档，将单词映射到实向量的技术称为词嵌入，现在的NLP基本都是使用词向量，根本原因是计算机无法直接理解人类语言，所以必须要先将自然语言“计算机化”，常见的选择就是向量化

!!!warning 为什么独立热向量是一个糟糕的选择
    独立热向量是指给每个不同单词一个单独编号，这种方法的最大问题是完全不能表达不同词之间的相似程度，而且维度很高

### word2vec
为了解决上述问题，引入word2vec，它能够将每个词映射到固定长度的向量，这些向量能够更好地表达不同词之间的相似性和类比关系，基本思想是上下文相似的两个词，它们的词向量也应该相近，比如苹果和梨都是常见的水果名称，它们会经常出现在相近的上下文中，那么它们的词向量就应该是相近的。

## BERT
在自然语言中上下文是很重要的信息，上面的模型对于一个词的编码都是固定的，但是同一个词在不同语境下的意思往往千差万别，因此催生了上下文敏感的模型，比如BERT，BERT基于Transformer的神经网络来理解和生成类似人类的语言，BERT采用仅编码器架构。在原始Transformer中既有编码器模块，也有解码器模块。传统的模型按顺序处理文本，一般从左到右，这种方法将感知范围限制在目标词之前的直接上下文中，BERT使用双向方法（B是bi），能够同时考虑左右上下文
