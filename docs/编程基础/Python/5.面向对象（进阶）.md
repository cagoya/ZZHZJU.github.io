# 面向对象（进阶）
## 接口和抽象基类
### 抽象基类
抽象基类是通过大鹅类型实现（非官方术语，用来描述一种更严格的类型检查机制）：
* 通过抽象方法（`@abstractmethod`）明确声明一个类需要实现哪些方法
* 子类必须实现抽象基类中定义的所有抽象方法，否则无法实例化
* 通过继承抽象基类，明确表示一个类实现了某个接口

比如：
```py
from abc import ABC, abstractmethod

# 定义抽象基类，不能直接实例化
class Animal(ABC):
    @abstractmethod
    def speak(self):
        pass
    # 还是可以定义实现好的方法
    def run(self):
        print("running")

# 实现抽象基类
class Dog(Animal):
    def speak(self):
        return "Woof!"

# 运行时类型检查
def check_animal(animal):
    if isinstance(animal, Animal):
        print(animal.speak())
    else:
        raise TypeError("Expected an Animal")

dog = Dog()
check_animal(dog)  # 输出: Woof!
```

### 混入类
在 Python 中，混入类（Mixin） 是一种用于代码复用的设计模式，主要用于提供额外功能，而不是单独实例化，本质就是多继承。
``` py
class LogMixin:
    def log(self, message):
        print(f"[LOG] {message}")

class User(LogMixin):
    def __init__(self, name):
        self.name = name
    
    def say_hello(self):
        self.log(f"User {self.name} says hello!")  # 来自 LogMixin
        print(f"Hello, I am {self.name}!")

# 测试
u = User("Alice")
u.say_hello()
```
### 限制绑定
之前也说了Python作为动态语言可以给实例绑定任何变量（对象），函数也是对象，所以其实也可以绑定新的函数，不过需要用`types`模块的`MethodType`对一般函数进行转换，把它变成相应对象的方法。此外，如果要给整个类绑定函数也是可以的。

但是如果我们想要限制绑定的属性（方法）怎么办，Python提供了一个特殊变量`__slot__`，可以用它来限制能添加的属性
```py
class Student(object):
    __slots__ = ('name', 'age') # 用tuple定义允许绑定的属性名称
```

### `@property`装饰器
标准库提供了一个`@property`装饰器可以让对象的方法像属性一样操作，这样能写出看上去更自然的代码。
```py
class Circle:
    def __init__(self, radius):
        self._radius = radius

    # 使radius可以像属性一样被操作
    @property
    def radius(self):
        return self._radius

    # 这个装饰器是上面那个 @property 生成的
    @radius.setter
    def radius(self, value):
        if value < 0:
            raise ValueError("Radius cannot be negative")
        self._radius = value

    @radius.deleter
    def radius(self):
        print("Deleting radius")
        del self._radius

# 使用
c = Circle(5)
print(c.radius)  # 输出: 5

c.radius = 10
print(c.radius)  # 输出: 10

del c.radius  # 输出: Deleting radius
```

## 类型提示
这是一个`3.5`之后才有的语法，目的是希望给python一个静态类型的系统，但是正如名字是“提示”而不是检查，Python始终是一门动态类型的语言。
### 基本类型提示
可以使用内置类型来标注变量和函数的类型
```py
# name应该是字符串，返回值也应该是字符串
def greet(name: str) -> str:
    return f"Hello,{name}"
```
### 复合类型提示
对于更复杂的类型，比如列表、字典，可以使用`typing`模块中的类型,`3.9`以上就不需这么做了，可以直接用内置类型（小写）
```py
from typing import List,Dict

# items应该是一个字符串列表，counts应该是一个键为字符串，值为整型的字典，返回值为None
def process_items(items: List[str],counts: Dict[str, int]) -> Node:
    pass
```
### 可选类型
有的时候，可以选择的参数未必只有一种，可以使用`Union`指定多种类型，`Optional`指定一种或者`None`
```py
from typing import Union, Optional

# 接受整数或者浮点数，返回浮点数或者返回None
def s(num: Union[int,float])->Optional[float]
```
### 类型别名
类似于C的`typedef`，Python也可以给类型取别名
```py
from typing import List
# Any表示任意类型
# 这里表示Vector是任意类型的列表
Vector=List[Any]
```
### 泛型
可以使用`TypeVar`来定义泛型，就如同C++的`template`
```py
from typing import TypeVar, List

T = TypeVar('T')

def first_item(items: List[T]) -> T:
    return items[0]
```
