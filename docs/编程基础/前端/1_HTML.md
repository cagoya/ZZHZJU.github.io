# HTML
html（hypertext mark language） 是一种文本控制语言，用标签区分不同模块
## 基础语法
``` html
<!DOCTYPE html>
<html>
<head>
    <title>我的第一个网页</title>
</head>
<body>
    <header>
        <h1>欢迎来到我的网页</h1>
    </header>
    <main>
        <p>这是一个段落，包含 <a href="#">一个链接</a>。</p>
        <img src="https://s1.imagehub.cc/images/2025/01/22/9af4be579b72c9ef3519418512dd3541.png" alt="示例图片">
    </main>
    <footer>
        <p>版权所有 &copy; 2025</p>
    </footer>
</body>
</html>

```
