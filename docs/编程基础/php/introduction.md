## 基础语法
PHP脚本在服务器上执行，然后以纯HTML结果发送回浏览器。PHP脚本可以放在文档中的任何位置，以`<?php`开始，`?>`结束，PHP是有短标签的，即`<?= ?>`，不过建议还是写标准写法。

PHP每一行都以分号结束，注释同C，使用`echo`和`print`输出，可以输出html语句，区别是`echo`可以输出一个或者多个字符串，其中的变量会自动替换。PHP的变量以`$`开始，并且是弱类型的语言，PHP要在一个函数中访问一个全局变量，需要使用 global 关键字。可以用`const`定义常量。

PHP支持以下几种数据类型：
* String 字符串
* Integer 整型
* Float 浮点型
* Boolean 布尔型
* Array 数组
* Object 对象
* NULL 空值

因为是弱类型语言，所以一般是用`===`

运算符同C，条件语句，`elseif`是连在一起的，循环有`foreach`

### 字符串
String没有重载 + ，而是有一个并置运算符`.`，`strlen()`会返回字符串的长度，`strpos()`可以查找一个指定文本

### 数组
PHP中使用`array()`创建数组，PHP中数组其实是数组和哈希表：
* 数值数组就是普通的数组，创建形如`$cars=array("Volvo","BMW","Toyota");`
* 关联数组其实是键值对，创建形如`$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");`

## 表单
当处理 HTML 表单时，PHP 能把来自 HTML 页面中的表单元素自动变成可供 PHP 脚本使用。
