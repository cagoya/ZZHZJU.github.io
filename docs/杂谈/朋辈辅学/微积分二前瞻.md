# 微积分二前瞻

如今的微积分二其实在以前是两门课程——微积分二和微积分三，如今合并了，所以内容稍微有点偏多，我所在的班级甚至去年期末之前补了一次课（当然，这个可能和放假调休有关。前年的微积分二期末特别难，主要是所有题都是中等偏上，连那个空间解析几何的题都很恶心，是斜着投影的，联立方程之后相当不好算，压轴我记得应该是三大公式结合场论的证明题；去年相对比较简单，印象中是各类积分考了一道题，然后是级数的证明压轴（不过我记得我应该是没做出来的。学好微积分对之后的概统、大物等数理课程和数值分析(NA)、机器学习(ML)等专业课都是很有帮助的。
