# Hyper-SD
## 简介
### 背景
扩散模型在推理过程中天然存在的多步数迭代去噪特性导致了较高的计算成本，所以扩散模型相关研究基本都聚焦在了如何减少步数，加速生成。

为了解决这些问题，字节跳动技术团队提出了一种名为 Hyper-SD 的轨迹分段一致性模型，该模型是一种新颖的扩散模型蒸馏框架，结合了轨迹保持蒸馏和轨迹重构蒸馏两种策略的优点，在压缩去噪步数的同时保持接近无损的性能，甚至可以压缩到一步。

### 蒸馏
!!!note 知识蒸馏 
    知识蒸馏(Knowledge Distillation)，是压缩模型的一种常用方法。
    
知识蒸馏是通过构建一个轻量化的小模型，利用性能更好的大模型的监督信息，来训练这个小模型，以期达到更好的性能和精度，这个大模型我们称之为teacher（教师模型），小模型我们称之为Student（学生模型）。

在早先的论文中已经有以下这两种蒸馏方法：
* 轨迹保持蒸馏（Trajectory Preserving Distillation，即TCD），尽可能保留教师模型生成过程中（即采样轨迹）的关键信息，确保学生模型生成结果的分布和动态过程与教师模型尽可能一致
* 轨迹重构蒸馏（Trajectory Reconstruction Distillation，即TRD），核心在于让学生模型重建出教师模型的整个生成轨迹，而不是仅仅模仿最终输出结果。

## 核心贡献
### TSCD
将整个时间步分成多个段，每段进行一致性蒸馏，并逐步合并，最终得到全时一致模型。这样做可以减少拟合难度、降低误差累积，提高低步数下的生成质量。

具体而言，该算法首先引入轨迹分段一致性蒸馏，在每个段内强制保持一致性，并逐渐减少段的数量以实现全时一致性。这一策略解决了由于模型拟合能力不足和推理误差累积导致的一致性模型性能次优的问题。

TSCD 多阶段训练策略：
* 将时间步分为 8、4、2、1 段依次训练。
* 使用 MSE 和对抗损失动态权重组合。
* 保持 ODE 轨迹一致性。

### 人类反馈学习
利用人类偏好（如美学得分、结构合理性）作为奖励信号，优化低步数生成模型的表现，使其更加符合人类主观审美和结构直觉。

人类反馈学习：
* 美学得分使用 LAION 和 ImageReward；
* 结构反馈使用实例分割模型（如 SOLO）判断结构合理性；
* 通过 LoRA 实现模块化合并，不改变输出域。

### 不同步数
引入分布匹配蒸馏（DMD）与得分蒸馏策略，强化一步生成能力，结合统一的 LoRA 插件支持 1~8 步推理，性能稳定。

### LoRA 插件
使用 LoRA（Low-Rank Adaptation）训练所有阶段的插件，实现了轻量、灵活的“即插即用”推理，适配多个基础模型（如 SDXL、SD15、anime 风格等）。
